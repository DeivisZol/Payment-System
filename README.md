# Payment-System

Objektinis programavimas JAVA Pratybu 3 uzduotis 2021 VU Informatika

3 lab. darbas
Mokėjimo sistema

Reikalavimai:
Turi būti panaudotos abstrakčios klasės (abstact class), interface'ai.
Sistemos reikalavimai: 
1. Sistema turi langus, kuriuose būtų galimybė suvesti studentų grupes ir tai grupei priskirtus studentus.
2. Sistemoje yra langas, kuriame įrašome mokėjimą kiekvienai grupei pasirinktą mėnesį (mokėjimas skirtingas kiekvieną mėnesį ir kiekvienai grupei).
3. Sistemoje galima gauti sąrašą studentų, matydami jų mokėjimus, pasirinktu mėnesiu ir šį sąrašą galime išsaugoti pdf formate.
4. Studentų mokėjimus galime, bet kada keisti (modifikuoti), saugoti.
5. Turi būti grafinė sąsaja (javafx, swing).
6. Studentus galime įtraukti su jų mokėjimais (atnaujinti mūsų kuriamą programą) iš csv ir Excel failo .
7. Studentus galime išsaugoti į csv ir Excel failą.
